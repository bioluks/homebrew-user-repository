# Bioluks Tap

## How do I install these formulae?

`brew install bioluks/tap/<formula>`

Or `brew tap bioluks/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bioluks/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
